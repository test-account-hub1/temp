# 모닝테이블 랜딩 페이지 발표 자료

## 0. 접속 페이지
https://temp-6pad6emiw-juneseok81-4659s-projects.vercel.app/


## 1. 프로젝트 한 줄 소개
감성 브런치 카페 **모닝테이블**의 브랜드 무드와 핵심 정보를 한 페이지에서 전달하는 반응형 랜딩 페이지입니다.

## 2. 기획 의도

### 브랜드 의도
- 따뜻한 우드톤과 크림 베이지 기반의 시각 언어로 "편안한 아침" 이미지를 전달
- 카페의 감성만 강조하지 않고, 실제 방문 전 필요한 정보(메뉴/공간/운영/예약)를 함께 제시

### 사용자 경험 의도
- 첫 화면에서 브랜드 인상 + 행동 유도(메뉴 보기, 예약 문의)까지 즉시 제공
- 스크롤 내에서 정보 탐색이 자연스럽게 이어지도록 섹션 순서를 설계
- 클릭 가능한 상세 정보로 "궁금한 사람은 더 깊게" 확인할 수 있도록 구성

## 3. 구현 방식

### 기술 선택
- **HTML + CSS only**
- 자바스크립트 없이도 인터랙션을 구현하도록 설계

### 정보 구조
- 섹션 구성: 히어로 → 대표 메뉴 → 매장 소개 → 공간 소개 → 운영 정보 → 예약 유도 → 푸터
- 상단 네비게이션 앵커 링크로 각 섹션으로 즉시 이동

### 핵심 구현 포인트
- `sticky header`: 스크롤 중에도 주요 메뉴 접근 유지
- 카드형 UI: 메뉴/공간/운영정보를 동일 패턴으로 제공해 스캔 효율 향상
- 히어로 캐러셀: 매장/음식 이미지를 자동 슬라이드로 노출해 시각적 집중도 강화
- 클릭 상세보기: `details/summary`로 메뉴/좌석 추가 정보 제공
- hover 인터랙션: 버튼/카드/이미지에 미세한 반응을 적용해 정적 페이지의 몰입도 향상

### 반응형 설계
- 주요 브레이크포인트: `980px`, `720px`
- 모바일에서 카드 레이아웃을 1열로 정리하고 가독성 우선 배치
- 모션 최소화 환경(`prefers-reduced-motion`) 대응
